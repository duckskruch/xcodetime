# xcodetime

AI-powered Health & Productivity Tracker built with SwiftUI.

## Features
- **Nutrition Tracking**: Add food by photo or text description using Gemini 2.0 Flash (via OpenRouter).
- **Quality Score**: AI evaluates your nutrition quality on a scale of 1-10.
- **Habit Tracking**: Manage your daily routines like Omega-3, Contrast Showers, and Cardio.
- **Planner**: Organize your tasks with AI assistance.
- **Local Data**: Powered by SwiftData for offline use and fast performance.

## Getting Started
1. Clone the repository.
2. Open `xcodetime.xcodeproj` in Xcode.
3. Open `xcodetime/Config.swift` and insert your **OpenRouter API Key**.
4. Run the app on a Simulator or Device.

## Requirements
- iOS 17.0+
- Xcode 15.0+
- OpenRouter API Key
